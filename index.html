projects

Fotofinder

Fotofinder was one of the first projects I created while at Turing. This project utilized HTML5, CSS, and jQuery to create cards that display a name, title, and an image that the user selects from their computer. The file selector will only allow image file types, to be selected. The application is designed to be responsive for mobile phones, tablets, and desktop devices.

HeadCount 2.0

HeadCount 2.0 was a paired project. This project incorporated React, HTML5, CSS, Javascript, JSON, and TDD. My partner and I utilized mock api data to display educational stats per district in Colorado. Testing was done with Enzyme and Jest. The app is responsive on phones, tablets, and desktop computers.

Swapi Box

Swapi Box was a paired project. We utilized HTML5, CSS/3, React, API calls, Javascript, JSON, and TDD. We drilled into the star wars api to display, people, planets, vehicles, and a host of other data. We allowed the user to select their favorite characters, planets and vehicles. Lastly, we created a scroll feature that streamed star wars movie data. Testing was done with Enzyme and Jest. The app is responsive.
